Event Triggers
  • Task assigned
    Notify assignee when a task is created or re-assigned.
  • Task status changed
    Notify assignee when a task moves to a different column.
  • New comment
    Notify assignee when someone comments on their task.
  • Project created
    Post to Teams when a new project is created.
  • Project status changed
    Post to Teams when a project's status is updated.
How it works
  • Teams messages are sent via an Incoming Webhook connector — no app registration required.
  • Email uses Office 365 SMTP with STARTTLS on port 587. The account must have SMTP AUTH enabled in Exchange Online.
  • All notifications fire in a background thread so they never slow down the UI.
  • The SMTP password is stored locally in instance/notify_config.json and is never committed to git.
